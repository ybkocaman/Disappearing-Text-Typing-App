# Disappearing Text Typing App

The Typing App is a simple Python application built using the Tkinter library that allows users to type text. If the user stops typing for 5 seconds, the text they typed will be cleared, and the countdown will reset.

## Features

- Users can type text in the provided text box.
- A countdown timer starts when the user stops typing.
- If the countdown timer reaches 5 seconds, the text is cleared.
- The countdown timer resets to 5 seconds every time the user types a new character.

## Getting Started

To run the Typing App, follow these steps:

1. Make sure you have Python installed on your system.

2. Clone this repository to your local machine or download the source code.

3. Open a terminal or command prompt and navigate to the project directory.

4. Run the following command to start the application:

   ```bash
   python main.py
